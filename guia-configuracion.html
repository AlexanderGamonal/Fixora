Guía de Configuración MVP Google Analytics 4 · Google Sheets
Proyecto Universitario – 2026
¿Para qué sirve esta guía? Te explica paso a paso cómo configurar Google Analytics 4 (para medir visitas) y Google Sheets (para registrar los datos del formulario), de forma que puedas medir con precisión los criterios de éxito de tu experimento MVP sin necesidad de un servidor propio.
1

Google Analytics 4 — Medir visitas

Qué mide: visitas totales, origen del tráfico, tiempo en página, dispositivos

1

Crear cuenta en GA4

Ingresa a analytics.google.com con tu cuenta Gmail. Haz clic en "Empezar a medir", escribe como nombre de cuenta FIXORA MVP y haz clic en "Siguiente".

2

Configurar la propiedad

  • Nombre de la propiedad: Landing Page FIXORA
  • Zona horaria: Perú (GMT-5)
  • Moneda: Sol peruano

Haz clic en "Siguiente" → "Siguiente" → "Crear".

3

Obtener tu ID de medición

En el Asistente de configuración, selecciona "Web". Ingresa la URL de tu página y haz clic en "Crear flujo". Aparecerá un código con este formato:

G-XXXXXXXXXX ← Copia este código
4

Activar el código en index.html

Busca el bloque comentado (alrededor de la línea 9) y:

  • Reemplaza las dos apariciones de TU_ID_AQUI con tu código G-XXXXXXXXXX.
  • Elimina los <!-- y --> que rodean el bloque.
<!-- Resultado final debe verse así: --> <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'G-XXXXXXXXXX'); </script>
💡
Una vez publicada la página, espera 24-48 horas para ver datos acumulados. Para datos en tiempo real, ve a Informes → Tiempo real en tu panel de GA4.
2

Google Sheets — Registrar datos del formulario

Qué mide: nombre, tipo (cliente/técnico), interés, servicio, distrito, preocupación

1

Crear la hoja de cálculo

Ve a sheets.google.com y crea una hoja nueva. Nómbrala FIXORA - Registros MVP. En la Fila 1, escribe estos encabezados en cada columna:

A: Fecha | B: Tipo | C: Nombre | D: Contacto E: Distrito | F: Servicio/Especialidad G: Preocupación/Experiencia | H: Interés
2

Abrir el Editor de Apps Script

Dentro de la hoja, ve al menú Extensiones → Apps Script. Borra todo el contenido por defecto y pega el siguiente código:

function doPost(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const params = e.parameter; const tipo = params.tipo_usuario || ''; const nombre = params.nombre || ''; const contacto = params.contacto || ''; const distrito = params.distrito || ''; const servicio = params.servicio_interes || params.especialidad || ''; const preocupacion = params.preocupacion || params.experiencia || ''; const interes = params.interes || ''; const fecha = new Date().toLocaleString('es-PE', { timeZone: 'America/Lima' }); sheet.appendRow([fecha, tipo, nombre, contacto, distrito, servicio, preocupacion, interes]); return ContentService .createTextOutput(JSON.stringify({ status: 'ok' })) .setMimeType(ContentService.MimeType.JSON); }

Guarda el proyecto con el nombre FIXORA Script (ícono 💾 o Ctrl+S).

3

Publicar el script como aplicación web

  • Haz clic en "Implementar" → "Nueva implementación".
  • Haz clic en el ícono ⚙️ → Selecciona "Aplicación web".
  • Ejecutar como: Yo (tu correo)
  • Quién tiene acceso: Cualquier persona ← Muy importante
  • Haz clic en "Implementar" y luego en "Autorizar acceso".
4

Copiar la URL generada

Aparecerá una pantalla con una URL de aplicación web de este estilo:

https://script.google.com/macros/s/AKfycby.../exec ← Cópiala completa
5

Pegar la URL en script.js

Abre el archivo script.js y busca esta línea al inicio del archivo:

const GOOGLE_SCRIPT_URL = 'URL_DE_TU_GOOGLE_APPS_SCRIPT_AQUI';

Reemplaza el texto entre comillas con la URL que copiaste en el paso anterior.

⚠️
Cada vez que re-publiques el script (por ejemplo, si haces cambios), Google te genera una nueva URL. Asegúrate de actualizar la variable GOOGLE_SCRIPT_URL en script.js con la URL más reciente.
